Hi, I'm Ewan Peebles! I'm a semi-experienced Minecraft: Bedrock edition modder. My current focus is an ElectronJS app that will manage Minecraft: Java Edition server. You can look at my repositories (I know they're messy) if you're bored and looking for a mod to play. It is quite uniquely called Java Server Manager, or JSM for short.
